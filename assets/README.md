## 🖼️ Image Guidelines

- **Resolution**: Minimum 1080p for screenshots
- **Format**: PNG for diagrams, JPG for photos
- **Naming**: Use descriptive, lowercase names with hyphens
- **Size**: Optimize for web (under 2MB per file)
